Le boîtier à code confidentiel

Un boîtier à code sert à n’autoriser l’accès à un endroit qu'aux seules personnes qui en possèdent le code.
Un tel dispositif peut également servir à limiter l’utilisation d’un appareil électrique, par exemple.
Les dispositifs qui sont présentés sur cette page ont été réalisés pour la cabane.

1) Présentation

1-1) Version 1 - 2001

Ce premier modèle ne comporte aucune pièce électronique. Il est constitué de quatre interrupteurs rotatifs, à huit positions chacun. Le bouton central est un bouton poussoir qui possède deux contacts indépendants, et qui se ferment quand quelqu’un appuie dessus pour valider le code qu’il a pu composer à l’aide des quatre interrupteurs rotatifs.

1-2) Version 2 - 2003

Ce second modèle est constitué d’un clavier, qui comporte neuf boutons poussoirs. Le dispositif contient également des relais et des temporisateurs, récupérés dans des automobiles. Lorsque les quatre chiffres du code ont été composés, celui-ci est validé automatiquement, au bout de trois secondes.

1-3) Version 3 - 2006

Ce troisième modèle possède désormais un clavier, qui comporte douze boutons poussoirs. Le code est programmable de façon électronique. Un buzzer génère un bref signal sonore, lorsqu’un bouton poussoir est appuyé, et une série de signaux sonores, lorsque le code qui vient d’être entré est incorrect.
