1) Présentation

Ce jeu est adapté du Mastermind, consistant à trouver une combinaison de quatre couleurs choisies parmi huit.
Dans le cas présent, la combinaison de couleurs est remplacée par un code à quatre chiffres.
La machine donne des indications au joueur sur le nombre de chiffres bien placés et mal placés.
Un chiffre proposé n'est compté qu'une seule fois dans le cas où la combinaison à découvrir le contient plus d'une fois, et les chiffres bien placés sont prioritaires par rapport aux chiffres mal placés.
Le joueur a onze essais. Il gagne le jeu s'il trouve le code avant d'avoir effectué ce nombre maximum d'essais.
Si le code n'a pas été trouvé à l'issue du nombre d'essais imparti, le jeu est perdu et le code qu'il fallait découvrir est affiché.
