1) Présentation

Vous aimez jouer à des jeux lorsque vous vous retrouvez à plusieurs lors de veillées ?

Si vous avez la chance d'avoir au moins un Minitel de disponible, vous pouvez bénéficier d'une version électronique moderne de jeux devenus de grands classiques !

↑ Je veux remonter ! | Liste

2) Les jeux : Aperçus

Les jeux auxquels il est possible de jouer avec ce projet sont :

  • Le 3615 BATAILLE NAVALE, se jouant à deux joueurs, donc avec deux Minitels équipés chacun d'une des réalisations ;
  • Le 3615 BATONNETS, se jouant seul contre la machine ;
  • Le 3615 MUNSTERMIND, se jouant seul ;
  • Le 3615 PENDU, se jouant seul.
