Réutilisation d'un module radio Hi-Fi

Réutilisation d'un module radio Hi-Fi.
Cet article présente le test d'un module récepteur radio, extrait d'une chaîne Hi-Fi qui ne fonctionne plus.

Le module radio couvre les gammes de fréquences suivantes : FM - 87,5 MHz - 108 MHz, grandes ondes - 144 kHz - 288 kHz, petites ondes - 525 kHz - 1605 kHz.
Il est géré par un circuit intégré de référence « TEA5762H », pilotable par des signaux numériques.
